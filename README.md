## 简介

深度学习（Deep Learning）正在以前所未有的速度重塑互联网及各行各业，其影响已深入搜索引擎、智能广告、医疗影像分析、个性化教育、精准农业和自动驾驶等领域。这项技术不仅开辟了全新的产品开发路径，也极大丰富了人们的生活体验。  

在学习深度学习的过程中，我尝试过大量教程和书籍，但作为零基础学习者，总感到学习过程充满阻碍。直到我遇到了吴恩达老师的 Coursera 深度学习课程，才发现可以如此系统、轻松地入门。
值得一提的是，我将课程中的 **TensorFlow 示例代码全部改写为 PyTorch 版本**，并对每一行代码进行了详细中文注释，方便读者理解与实践。本书及相关代码均可在我的 GitHub 下载：[GitHub 仓库链接](https://github.com/Luxianyu/DeepLearning-Notes)

本书旨在帮助像我一样的学习者：
- 系统掌握深度学习核心理论  
- 熟练应用 PyTorch 完成实践任务  
- 为未来的 AI 项目打下坚实基础

## 第一部分 神经网络和深度学习

本部分深入讲解神经网络基本概念，包括浅层网络与深度网络的构建方法，以及训练数据的方法。完成本部分后，你将能够使用深度神经网络完成猫咪图像识别等任务。

---

## 第二部分 深层神经网络优化

本部分聚焦深度学习模型的优化与调试，你将学习：
- 超参数调优  
- 正则化方法  
- 偏差与方差诊断  
- 高级优化算法：Momentum、Adam  

通过这些技能，你的神经网络性能将得到显著提升。

---

## 第三部分 结构化机器学习项目

本部分引导你构建结构化深度学习工程，包括：
- 训练集、验证集、测试集划分  
- 处理训练与测试数据分布不一致  
- 端到端深度学习原理与应用场景  
- 大量真实项目最佳实践

## 第四部分 卷积神经网络（CNN）

本部分重点讲解卷积神经网络及其在图像处理中的应用，帮助你掌握 CNN 模型构建技能，为视觉任务打下坚实基础。

---

## 第五部分 序列模型

本部分学习序列模型，包括：
- 循环神经网络（RNN）  
- 长短期记忆网络（LSTM）  
- 注意力机制  
- Transformer  

应用场景涵盖自然语言处理（NLP）、语音识别、音乐创作等领域，探索序列数据的无限可能。
